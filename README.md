# Flora Backend

Backend API for the Flora flower shop application.

## Technologies

- Node.js
- Express.js
- PostgreSQL
- Sequelize
- Render

## Live API

https://flora-backend-v936.onrender.com

## Swagger

https://flora-backend-v936.onrender.com/api/docs

## API Endpoints

- GET /api/bouquets
- GET /api/bouquets/:id
- POST /api/bouquets
- PATCH /api/bouquets/:id
- DELETE /api/bouquets/:id
